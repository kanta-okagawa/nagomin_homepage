- React 18 + TypeScript による SPA 構成

- Comic Sans フォントによる親しみやすいデザイン

- CTAボタン: 黄色背景（`bg-yellow-500`）でホバー効果付き

### ファイル構成

## 変更履歴の記録方針

このプロジェクトでは以下の形式で変更履歴を記録します：

### 変更の種類
- **追加** - 新機能の追加
- **変更** - 既存機能の変更
- **廃止** - 今後削除される機能
- **削除** - 削除された機能
- **修正** - バグ修正
- **セキュリティ** - セキュリティに関する変更

### バージョニング
セマンティックバージョニング（SemVer）に準拠：
- **MAJOR**: 互換性のない API の変更
- **MINOR**: 後方互換性がある機能性の追加
- **PATCH**: 後方互換性があるバグ修正
